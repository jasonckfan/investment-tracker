圖標文件說明

請使用以下工具生成PNG圖標：
1. 訪問 https://favicon.io/favicon-converter/
2. 上傳SVG圖標或使用在線工具生成
3. 下載各尺寸的PNG圖標
4. 重命名為 icon-{size}x{size}.png

或使用命令行工具：
- ImageMagick: convert -background none icon.svg icon.png
- Inkscape: inkscape -w 192 -h 192 icon.svg -o icon-192x192.png

需要的尺寸：72x72, 96x96, 128x128, 144x144, 152x152, 192x192, 384x384, 512x512
