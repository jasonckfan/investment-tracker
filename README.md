# 投資計劃追蹤 PWA

追蹤您的三劍客投資計劃表現，提供再平衡提醒和績效分析。

## 功能特點

- 📊 實時追蹤投資組合表現
- 📈 資產增長趨勢圖表
- ⚠️ 再平衡提醒（每半年）
- 📱 PWA支持（可安裝到手機/桌面）
- 🔔 推送通知提醒
- 📅 重要里程碑時間線

## 投資計劃詳情

- **計劃名稱**: 三劍客精簡版
- **開始日期**: 2025年5月4日

## 安裝與使用

### 1. 本地開發

```bash
# 克隆倉庫
git clone https://github.com/jasonckfan/investment-tracker.git
cd investment-tracker

# 啟動本地服務器
python3 -m http.server 8000

# 訪問 http://localhost:8000
```

### 2. 更新數據

```bash
# 運行數據更新腳本
python3 update_data.py

# 數據將保存到 data/portfolio.json
```

### 3. 部署到GitHub Pages

```bash
# 提交更改
git add .
git commit -m "Update investment data"
git push origin master

# GitHub Pages將自動部署
```

## PWA安裝

### Android (Chrome)
1. 訪問網站
2. 點擊菜單 → "添加到主屏幕"
3. 確認安裝

### iOS (Safari)
1. 訪問網站
2. 點擊分享按鈕
3. 選擇"添加到主屏幕"

### 桌面 (Chrome/Edge)
1. 訪問網站
2. 點擊地址欄右側的安裝圖標
3. 確認安裝

## 再平衡規則

- **頻率**: 每半年檢視一次（5月、11月）
- **觸發條件**: 任一ETF偏離目標配置超過±15%
- **操作**: 賣出超配ETF，買入低配ETF

- HTML5 + CSS3 + JavaScript

## 注意事項

1. **風險提示**: 此應用僅供參考，不構成投資建議
2. **數據延遲**: 價格數據可能有延遲，請以券商數據為準
3. **稅務考量**: 投資收益可能涉及稅務，請諮詢專業人士
4. **過往回報**: 過去表現不代表未來回報
